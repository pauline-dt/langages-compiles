# TP2 - Le processus de compilation et les bases du C

Ce dossier contient les exercices demandés dans le TP2.

## Compiler tous les programmes corrects

```bash
make
```

## Lancer un programme

Exemple :

```bash
./convertisseur
```

## Nettoyer les exécutables

```bash
make clean
```

Les fichiers du dossier `erreurs` contiennent volontairement des erreurs et ne sont pas compilés par le Makefile.
